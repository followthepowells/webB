Mapping Land Development Factors

Problem Background

Land development in Toronto can depend on many factors when deciding where to start looking to start the process. This data is not consolidated into one source and the sources that are out there are not presented in the most comprehensive manner. With the advent of new GIS tools at the disposal of GIS professionals, it is more achievable than ever to create comprehensive and easy to use information web maps to present this information in its entirety to the public or interested individuals and companies.

Ideal

A mobile-friendly interactive web map that compares and visualizes different neighborhoods according to education, crime rates, income, and housing costs which can help developers, city planners, non-government officials identify potential areas of concern.

Reality

There is no mobile-friendly and interactive web map that displays, aggregate, and compare these factors together.

Consequences

It is hard for users to visualize areas using tabular data directly available from Toronto’s Open Data Portals. Therefore, the identification of potential locations will be slow.

Proposal

Create an interactive web map that displays the 4 factors above where comparison tools or charts are present per neighborhood.

Technology Used

FME, ArcGIS Pro, ArcGIS Experience Builder, ArcGIS Dashboard and ArcGIS StoryMap (if time permits)
